# CronoPay

**AI-Powered Payment Agent for Cronos Blockchain**

## Description

CronoPay is an MCP-native payment agent that brings AI intelligence, visual planning, and developer tools to the Cronos ecosystem. The first payment agent with explainability, transaction simulation, and multi-step conditional logic.

## What It Does

- **AI Planning**: Multi-step execution plans with risk assessment and explainability
- **Conditional Payments**: Market-aware transfers ("Send if CRO > $0.10")
- **Transaction Simulator**: Risk-free dry run before execution
- **Batch Transfers**: Airdrop to multiple recipients in one operation
- **Visual Planning**: HTML/Mermaid/ASCII execution flow diagrams
- **Smart Contract Tools**: Inspect, read, and estimate gas via natural language
- **Transaction Analytics**: Query history with AI-generated insights
- **MCP-Native**: Works with ChatGPT, Claude, and any MCP client
